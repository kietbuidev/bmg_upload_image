# Cloudinary Upload API (Express + TS) - Railway

## ENV
Sao chép `.env.example` thành `.env` (local) và điền:
- CLOUDINARY_CLOUD_NAME
- CLOUDINARY_API_KEY
- CLOUDINARY_API_SECRET
- (tuỳ chọn) CLOUDINARY_FOLDER
- PORT (mặc định 8080)

## Development
```bash
npm i
npm run dev
# mở http://localhost:8080/docs để test form
